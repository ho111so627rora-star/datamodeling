# 採用選考管理システム

データモデリング最終課題として作成した、企業の採用情報を管理する Web アプリケーションです。

## 使用環境

- Python 3.13
- Flask 3.1.2
- SQLite3
- データベース接続には Python 標準の `sqlite3` を使用

## 初期化

```sh
python init_db.py
```

## ローカル起動

```sh
python app.py
```

ブラウザで `http://127.0.0.1:5000` を開きます。

## 主な機能

- 採用状況ダッシュボード
- 候補者と応募の検索
- 候補者の登録・編集
- 年度別の応募登録・更新・削除
- 面接と評価の登録
- ステージ別件数の集計
